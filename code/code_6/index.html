Union of two arrays

2021-04-09 19:00:29.662601

Given two arrays a[] and b[] of size n and n respectively. The task is to find union between these two arrays. Union of the two arrays can be defined as the set containing distinct elements from both the arrays. If there are repetitions, then only one occurrence of element should be printed in union. 

            
            
int doUnion(int a[], int n, int b[], int m)  {
    set s;
    for(int i = 0; i < n; i++){
        s.insert(a[i]);
    }
    for(int i = 0; i < m; i++){
        s.insert(b[i]);
    }
    return s.size();
}
